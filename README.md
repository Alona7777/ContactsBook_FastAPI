# Get started

1) Clone repository:

    ``` https://github.com/Alona7777/ContactsBook_FastAPI.git```

2) Activate Poetry environment:

    ```poetry shell```

3) Install packages:

    ```poetry install```

4) Run docker compose:
    ```docker compose up -d```

5) Run migrations:
    ```alembic upgrade heads```

6) Run main.py
